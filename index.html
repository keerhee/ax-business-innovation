커리큘럼 개요

n8n 워크플로우와 LLM(OpenAI), 벡터DB(Qdrant)를 활용해 은행 실무에서 반복되는 문서 분류·여신심사·결재·고객응대 업무를 자동화하는 실습 중심 커리큘럼입니다. 매주 실습 시트(.xlsx)·워크플로우 정의(.json)·이메일/Slack 예시 이미지가 함께 제공됩니다.

n8n Google Sheets / Gmail / Calendar Slack Webhook OpenAI Qdrant (RAG) Docker Self-host

학습 방법

  1. 먼저 사전 준비 Tip 으로 n8n / Google OAuth / Slack Webhook / OpenAI API Key / Qdrant 환경을 갖춥니다.
  2. 각 주차의 실습가이드 HTML을 열어 흐름을 먼저 이해합니다.
  3. 해당 주차 폴더의 W*.json을 n8n에 import → 시트(.xlsx)와 예시 이미지로 동작을 검증합니다.
  4. 막히는 부분이 있으면 본 매뉴얼의 색상 코딩 시스템(🔵 입력 / 🟢 처리 / 🟡 분기 / 🔴 알림 / 🟣 외부 API)을 따라 노드를 점검합니다.
