Tool Introduction

n8n이란?

n8n은 "만약 ~이 발생하면, ~을 한다"는 규칙을 클릭으로 만드는 도구입니다. 코딩 없이 수백 개의 앱·서비스를 연결해 반복 업무를 자동화합니다.

쉽게 이해하기: 엑셀 매크로처럼 반복 작업을 자동화하는데, 엑셀 안이 아니라 앱과 앱 사이를 연결합니다. 예: "구글 시트에 새 행이 생기면 → 자동으로 이메일 발송"
🔌 트리거 (Trigger)
자동화를 시작시키는 신호.
은행 예시: 여신 신청 시트 새 행 추가 / 매일 오전 8시 / 연체 고객 이메일 수신 / 이상거래 감지 Webhook
⚙️ 액션 (Action)
트리거 이후 실행할 작업.
은행 예시: 접수 확인 이메일 발송 / 심사역 캘린더 일정 등록 / 팀장 Slack DM / 규정 조항 검색
🔀 조건 분기 (IF/Switch)
상황에 따라 다른 경로 선택.
은행 예시: 여신 1억 미만 → 자동승인 / 1억~10억 → 팀장승인 / 10억↑ → 본부위임
🤖 AI Agent Node
Claude AI를 워크플로 안에 내장.
은행 예시: 재무제표 위험도 요약 / 민원 유형 자동 분류 / 만기 고객 맞춤 안내문 생성

RAG가 뭔가요?
Retrieval-Augmented Generation — "찾아서 읽고 답하기". 규정집·매뉴얼·보고서를 AI가 직접 검색해 근거 있는 답변을 생성합니다. "여신취급세칙 제23조의 내용은?" 같은 질문에 정확한 출처와 함께 답합니다. n8n이 이 전 과정을 자동으로 연결합니다.

Google Opal이란?
"보고서 요약기 만들어줘"라고 입력하면 Gemini AI가 실제 동작하는 웹 앱을 자동 생성하는 Google Labs 실험 도구. 완성된 앱은 링크로 팀원과 바로 공유 가능. n8n을 충분히 익힌 지금, Opal로 빠른 AI 앱 프로토타이핑을 추가 학습합니다.
